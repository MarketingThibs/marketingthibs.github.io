Unsub Page Builder

Generate a branded unusbcription page for SFMC.

This tool will generate HTML + CSS + AMPscript code for an unsubscription page.
This will allow you to brand : the logo, colors, wording. Besides, the AMPscript code acts like a loop. Here is the functionnal workflow :
  1. Email is received
  2. The subscriber clicks on the unsubscription link
  3. The unsub page updates instantly all subscribers list status + Salesforce lead and contact object
  4. You can resubscribe by clicking on the resub button

AMPscript code snippet

Visual Studio Code extension for AMPscript

To help you coding with AMPscript for Salesforce Marketing Cloud : as you type get proposed system strings and other functions.
How to ?
  1. When start typing the AMPscript functions and system strings names, press enter to auto-complete your code ampscript-snippets
  2. Press cmd + shift + p and then >insert snippet to review all the snippets available.
  3. Press enter to insert one ampscript-snippets
  4. When previewing the snippets, click on i to see the whole description of the different functions

SMFC - DE Inspector & CSV data export

Search all the data extension from your MC account as you type. Diplays information about the DE such as the folder Id, or the fields infomations. There is also a .csv option for exporting records. You can host this into a cloud page.

MC Tracking Import In Google Sheets

The idea here is to use Marketing Cloud Cloud Pages to display email tracking according the URL parameters that we fill. From a Google Sheet we can import the JSON from this URL. In other words we will be using Cloud Page as a JSON feed.

Automation - Update SC from MC

Marketing Cloud connect does not support bidirectional synchronisation, but we can use AMPscript Salesforce Functions to run into a script activity in the Automation Studio.

SSJS - Data Views Clones

Create Marketing Cloud Data Views clones - Two scripts (SSJS) that clones data extensions from the data views and create the queries that populate them.
